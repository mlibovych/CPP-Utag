### Tech/framework used
​

​

​
#### 2.GUI library
​

​
#### 3. External Libraries
​
 [Taglib](https://taglib.org)

### Installation:
    1. git clone --recursive https://github.com/mlibovych/CPP-Utag
    2. cmake . -Bbuild -Wdev -Werror=dev && cmake --build ./build

### How to use?
    ./utag [directory]
